Se está creando una aplicación que va a estar conectada con un prototipo que permita almacenar contactos telefónicos en el dispositivo. Para ello cada contacto debe contener nombre completo, teléfono y organización. Se requiere que la aplicación permita añadir 3 contactos verificando que el número no esté almacenado, buscar contactos almacenados y eliminar contactos si el usuario lo requiere. Recuerde que el sistema debe terminar cuando el usuario así lo indique. 

                    
                    Proceso taller_5_ejercicio_ciclos_6
                        Definir nombre, organizacion Como Caracter;
                        Dimension nombre[3], organizacion[3];
                        Definir telefono,opcion,opcion_contactos, contador, opcion_eliminar, inicio,x Como Entero;
                        Dimension telefono[3];
                        contador <- 0;
                        para x <- 0 Hasta 2 Con Paso 1 Hacer
                            nombre[x] <- 'No existe';
                            telefono[x] <- 0;
                            organizacion[x] <- 'No existe';
                            
                        FinPara
                        
                        Repetir
                            
                            Escribir '1. Crear nuevo contacto';
                            Escribir '2. Buscar contactos ';
                            Escribir '3. Eliminar contactos';
                            Escribir '4. Salir';
                            leer opcion;
                            Segun opcion Hacer
                                1:
                                    Escribir 'Solo podra guardar 3 contactos';
                                    Si contador < 3  Entonces
                                        nombre[contador] <- escribirnombre ('Ingrese el nombre de contacto');
                                        telefono[contador] <- escribirtelefono('Ingrese el numero de telefono');
                                        organizacion[contador] <- escribirnombre('Ingrese el nombre de la organizacion');
                                    SiNo
                                        Escribir 'Agenda llena';
                                    FinSi
                                    contador <- contador +1;
                                    Escribir  contador;
                                2:
                                    lectorAgenda(contador,nombre,telefono,organizacion);
                                3:
                                    Escribir 'Ingrese el numero del contacto que desea eliminar';
                                    Leer opcion_eliminar;
                                    Eliminarcarac(opcion_eliminar,nombre);
                                    Eliminarcarac(opcion_eliminar,organizacion);
                                    Eliminarnum(opcion_eliminar,telefono);
                            FinSegun
                        Hasta Que opcion=4;
                        
                    FinProceso 
                    
                    SubProceso nombre <- escribirnombre(mensaje)
                        Definir nombre Como Caracter;
                        Escribir mensaje;
                        Leer nombre;
                    FinSubProceso
                    
                    SubProceso nombre <- escribirtelefono(mensaje)
                        Definir nombre Como entero;
                        Escribir mensaje;
                        Leer nombre;
                    FinSubProceso
                    
                    SubProceso lectorAgenda(variable1,variable2,variable3,variable4)
                        Definir a Como Entero;
                        a <- 0;
                        Para a<-0 Hasta 2 Con Paso 1 Hacer
                            Escribir variable2[a]; 
                            Escribir variable3[a];
                            Escribir variable4[a];
                        FinPara
                        
                    FinSubProceso
                    
                    SubProceso Eliminarcarac(variable1,variable2)
                        
                        variable2[variable1] <- 'Inexistente';
                        
                    FinSubProceso
                    
                    SubProceso Eliminarnum(variable1,variable2)
                        
                        variable2[variable1] <- 0;
                        
                    FinSubProceso

El parqueadero "El guardián" presta sus servicios de parqueadero nocturno para los usuarios del barrio y requiere una aplicación que permita registrar los vehículos que se cuidan en estas instalaciones. Se sugiere que el parqueadero tenga los atributos del vehículo como son, placa y marca, y los datos del cliente como son nombre completo y número de teléfono. Para cada vehículo se debe permitir la opción de ingresar al parqueadero, retirar del parqueadero y consultar si un vehículo se encuentra en el parqueadero. Recuerde que el sistema debe terminar cuando el usuario así lo indique. Tenga en presente que el parqueadero solo puede almacenar máximo 5 vehículos.

                            nombre[x] <- 'No existe';

                            marca[x] <- 'No existe';
                        FinPara
                        
                        Repetir
                            Escribir 'Bienvenido a El Guardian, seleccione alguna de las opciones a continuacion';
                            Escribir '1. Ingresar datos de vehiculos';
                            Escribir '2. consulta de vehiculo';
                            Escribir '3. Retiro de vehiculo';
                            Escribir '4. Salir';
                            leer opcion;
                            Segun opcion Hacer
                                1:
                                    Escribir 'ingrese los datos de su vehiculo';
                                    Si contador < 5  Entonces
                                        nombre[contador] <- escribirnombre ('Ingrese el nombre de contacto');
                                        telefono[contador] <- escribirtelefono('Ingrese el numero de telefono');

                                    contador <- contador +1;
                                    Escribir  contador;
                                2:
                                    lectorgarage(contador,nombre,telefono,placa,marca);
                                3:
                                    Escribir 'Ingrese el numero del vehiculo que desea eliminar';
                                    Leer opcion_eliminar;
                                    Si (opcion_eliminar > 0) Y (opcion_eliminar < 6) Entonces
                                        Eliminarcarac(opcion_eliminar,nombre);
                                        Eliminarcarac(opcion_eliminar,marca);
                                        Eliminarcarac(opcion_eliminar,placa);
                                        Eliminarnum(opcion_eliminar,telefono);
